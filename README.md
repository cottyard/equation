# Equation Solver Training

This is a web-based tool to train students in solving systems of linear equations.

## Features
- Generate random systems of linear equations (1 to 4 variables).
- Supports rational numbers (integers and fractions).
- Pretty math rendering using MathJax.
- Interactive operations:
    - Add/Subtract/Multiply/Divide by a term.
    - Add/Subtract equations.
    - Flip equation sides.
- Win condition detection.

## How to Run
Simply open `index.html` in a modern web browser.

## Technologies
- HTML/CSS/JavaScript
- [math.js](https://mathjs.org/) for symbolic computation.
- [MathJax](https://www.mathjax.org/) for rendering.
